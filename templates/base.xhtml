GAE - AppEngine Wrapper Framework

O GAE é um framework que tem como objetivo facilitar imensamente o desenvolvimento de aplicações web com o AppEngine através da utilização de bibliotecas já disponíveis na internet assim como receitas e boas práticas visíveis em outros projetos. Muito deste framework é baseado no bloog, um sistema de blog para o AppEngine.
